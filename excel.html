📗 직장인 필수 엑셀(Excel) 가이드

엑셀 작업 시간을 절반으로 줄여주는 핵심 노하우를 공개합니다.

1. 필수 단축키 (Best 5)

기능 단축키 설명
범위 선택 Ctrl + Shift + 방향키 데이터가 있는 끝까지 범위를 즉시 선택합니다.
표 만들기 Ctrl + T 선택된 범위를 즉시 데이터 표로 변환합니다.
값만 붙여넣기 Alt + E + S + V 수식 제외 데이터 값만 붙여넣을 때 유용합니다.
필터 설정 Ctrl + Shift + L 데이터 필터를 즉시 켜고 끕니다.
오늘 날짜 입력 Ctrl + ; 현재 날짜를 즉시 입력합니다.

2. 반드시 알아야 할 필수 함수

  • VLOOKUP: 다른 표에서 특정 값을 찾아올 때 사용 (가장 빈번)
  • IF: 조건에 따라 다른 결과를 표시할 때 사용
  • SUMIF / COUNTIF: 특정 조건에 맞는 합계나 개수를 구할 때 사용
  • TEXTJOIN: 여러 셀의 텍스트를 기호와 함께 하나로 합칠 때 사용
  • XLOOKUP: VLOOKUP의 한계를 넘는 최신형 조회 함수

3. 데이터 정리 팁

엑셀 작업 전, 데이터 형식을 통일하는 것이 수식 오류를 줄이는 첫 번째 단계입니다.

  • 날짜 형식은 반드시 YYYY-MM-DD 형태로 통일하세요.
  • 숫자 데이터에 문자가 섞여 있지 않은지 확인하세요.
  • '중복된 항목 제거' 기능을 활용해 데이터를 정제하세요.
